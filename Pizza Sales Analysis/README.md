# Pizza Sales Analysis (Excel + SQL)

## Project Description
This project analyzes pizza restaurant sales data using Microsoft SQL Server and Microsoft Excel. SQL was used to calculate key business metrics and trends, while Excel was used to design an interactive dashboard for business decision-making.

## Tools Used
- Microsoft SQL Server (SSMS)
- Microsoft Excel
- Data Cleaning & Analysis
- Dashboard & Data Visualization

## SQL Analysis
SQL queries were written to calculate revenue, order trends, product performance, and customer behavior. All SQL scripts are available in the `sql` folder.

## Excel Dashboard
An interactive Excel dashboard was created to visualize KPIs, sales trends, and product performance.
